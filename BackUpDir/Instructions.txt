1. Reset proxy & clean install
	Purged old Squid: sudo apt purge squid -y
	Installed fresh Squid: sudo apt install squid -y
	Verified squid.conf exists and service is running
2. Created backup
	sudo cp /etc/squid/squid.conf /etc/squid/squid.conf.backup
3. Created block list
	sudo nano /etc/squid/blocked_sites.txt
	Added facebook.com
4. Linked block list in Squid
	Edited /etc/squid/squid.conf:

	acl blocked_sites dstdomain "/etc/squid/blocked_sites.txt"
	http_access deny blocked_sites

5.Reloaded Squid
	sudo systemctl reload squid
